# 한 주에 한 문제

> 📌 **챌린지 규칙**
>
> - **벌칙:** 3,000원 이하 음료 돌리기  
> - **월요일에 문제 결정**
> - **날먹 문제 금지, 도전할만한 문제로 선정하기**
> - **일요일까지 코드 작성 완료!**

## 소개

친구들과 함께 매주 백준에서 도전할 만한 문제를 선정해 풀이하는 챌린지입니다!  

## 진행 방식

1. **문제 선정**  
   - 매주 월요일, 이번 주에 풀 문제를 선정합니다.
2. **문제 풀이**  
   - 각자 자신의 폴더에 풀이 코드를 업로드합니다.  
     (예: `./닉네임/문제번호.py`)
3. **코드 리뷰**  
   - 서로의 코드를 읽고, 피드백이나 질문을 남깁니다.
4. **벌칙**  
   - 기한 내에 제출하지 않으면 3,000원 이하 음료를 돌립니다!
